# KolPages - רדמי

KolPages היא אפליקציית שולחן עבודה (WPF) להצגת דפי אינטרנט בסביבה מותאמת. היא מאפשרת ניווט לאתרים ושומרת הגדרות בסיסיות בקובץ תצורה כדי להתאים את ההתנהגות ללא שינוי קוד.

## קובץ appsettings.json
הקובץ appsettings.json נמצא בשורש הפרויקט ומשמש להגדרת פרמטרים שניתן לשנות ללא קימפול מחדש. שינוי הערכים בקובץ מאפשר להתאים את התנהגות האפליקציה לסביבה או לצרכים שונים.

### איך משתמשים
1. פתח את הקובץ appsettings.json.
2. ערוך את הערכים הרצויים בהתאם לשמות המפתחות.
3. שמור את הקובץ והפעל מחדש את האפליקציה כדי שהשינויים ייכנסו לתוקף.

### הערות
- מומלץ לשמור גיבוי של הקובץ לפני שינוי משמעותי.
- אם שינוי מסוים לא משפיע, ודא שהאפליקציה הופעלה מחדש.
